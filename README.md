## Run Locally

Clone the project

```bash
  git clone https://github.com/codenamezaxx/ReactJs-Online-Quran
```

Go to the project directory

```bash
  cd ReactJs-Online-Quran
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React, W3.css

**Server:** Node, Vercel
